▶본 수강신청

    • 🐾수강신청(수강희망과목) : 수강희망과목 내역을 조회하여 바로 신청
    🐾 수강신청(학부) : 수강신청 대상과목들을 학과별로 모두 조회
    ※ 수강신청 시간이 많이 소요됨
    🐾수강신청(간편) : 교과목 코드-분반을 직접 입력
    ※ 수강신청(학부/대학원) 보다 빠르게 수강신청 가능
    🐾 수강신청(교과목명) : 개설 교과목명을 검색
    • 수강신청(수강희망과목) → 해당 교과목번호 선택 → 신청
    • 해당 교과목번호를 클릭하면 수강희망과목 확정 팝업이 나옴
    • 확인 메시지창에서 [확인]버튼을 클릭 하면, 문자열 입력 화면으로 넘어감
    ※초기 전산망 서버 과부하 방지를 위해 9시 15분까지는 [확인] 버튼 클릭 시 교과목 수강신청 완료(문자열 입력화면 생략)
    • 수강신청내역 화면에서 [Delete]버튼 클릭하여, 수강신청내역 삭제도 가능
